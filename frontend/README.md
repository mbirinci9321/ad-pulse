# Frontend - AD User Management Interface

React + TypeScript + Tailwind CSS ile geliştirilmiş modern kullanıcı arayüzü.

## Kurulum

```bash
npm install
```

## Geliştirme Modu

```bash
npm run dev
```

Uygulama `http://localhost:3000` adresinde çalışacaktır.

## Build

```bash
npm run build
```

Build çıktısı `dist` klasöründe oluşturulacaktır.

## Özellikler

- Modern, responsive tasarım
- Kullanıcı listesi ve arama
- Grup bazlı filtreleme
- Kullanıcı detay görünümü
- Şifre sıfırlama
- Hesap durumu yönetimi
- Grup üyelik yönetimi

## Teknolojiler

- React 18
- TypeScript
- Tailwind CSS
- React Router
- Axios
- Heroicons
